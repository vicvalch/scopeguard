# Realtime Operational Telemetry Runtime

This runtime adds deterministic, governance-scoped operational telemetry cognition.

## Core design
- Continuous signal ingestion with scope isolation.
- Material operational delta detection.
- Significance scoring and suppression to prevent telemetry spam.
- Drift and survivability movement tracking.
- Escalation, PM overload, propagation, topology, and stabilization telemetry.
- Operational pulse as live heartbeat.
- Telemetry-driven organizational twin evolution input via deterministic APIs.

## Anti-noise safeguards
- Thresholded delta materiality.
- Deduplicated retained delta metrics.
- Suppression diagnostics exposed in runtime result.

## Governance controls
- company/workspace scope required.
- no cross-tenant synthesis.
- deterministic templates for narratives and alerts.
