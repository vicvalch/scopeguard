# Organizational Digital Twin Runtime

This runtime introduces deterministic organizational simulation infrastructure built from operational evidence rather than synthetic simulation theater.

## Philosophy
- Deterministic, evidence-bound simulation only.
- Bounded confidence with explicit uncertainty and causality.
- Governance-safe, tenant/workspace isolated computation.

## Capabilities
- Organizational topology graph generation.
- Operational state synthesis and state transitions.
- Propagation simulation and survivability simulation.
- Intervention and stabilization trajectory simulation.
- Fragility modeling, scenario projections, evolution snapshots.
- War-room twin context, diagnostics, and deterministic narratives.

## Anti-theater safeguards
- Every output carries evidence, confidence, uncertainty, causality rationale, survivability rationale, and governance boundaries.
- No black-box scoring and no fabricated organizational entities.
- Simulation remains bounded to available operational memory.
