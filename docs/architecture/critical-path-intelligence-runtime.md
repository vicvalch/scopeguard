# Critical Path Intelligence Runtime

This runtime adds deterministic temporal operational cognition to PMFreak operational memory.

## Capabilities
- Dependency graph assembly from scoped operational records.
- Critical execution chain detection using pressure-weighted dependency concentration.
- Temporal propagation windows (`immediate`, `near_term`, `medium_term`, `long_term`).
- Milestone survivability and collapse likelihood modeling.
- Hidden dependency inference with bounded confidence and explicit uncertainty.
- Bottleneck, fragility, instability, and survivability forecast outputs.
- Deterministic narratives and diagnostics with evidence and causality chains.

## Anti-hallucination safeguards
- Evidence-first outputs for every inferred object.
- Confidence dampening hooks and uncertainty fields preserved in all result contracts.
- Hidden dependency inference requires cross-signal co-occurrence (governance + milestone pressure).
- No calendar date hallucination; propagation timing is represented with temporal bands.

## Isolation and governance
- Scope-bound requests and company-required guardrail in governance assertion.
- Runtime manager APIs operate through scoped context only.
