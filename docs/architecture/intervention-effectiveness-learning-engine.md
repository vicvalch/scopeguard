# Intervention Effectiveness Learning Engine

Deterministic operational learning layer for outcome-driven intervention adaptation.

- Bounded adaptation only (0.3-0.7 prioritization guardrails).
- Explainable evidence + confidence + uncertainty on each output.
- Reversible calibration adjustments with governance-safe limits.
- Failure and drift signal detection to prevent stale recommendations.
- Tenant/workspace/project scoping preserved in request contracts.
